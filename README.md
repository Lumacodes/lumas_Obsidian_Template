# lums's Vault

A personal Obsidian vault for reading, tracking books, and organizing notes.

## Overview

This vault is set up for:
- Managing a book library with Bookshelf
- Keeping book notes with metadata like title, author, genre, and cover
- Organizing reading notes, courses, and general content
- Using daily notes for journaling and planning

## Structure

- Bookshelf.md — main reading hub note
- Reading/Books/ — book notes and cover images
- Templates/ — note templates for books, courses, meetings, and daily notes
- Extra/Diary/ — daily journal notes
- Scripts/ — automation for fetching book covers

## Features

- Bookshelf plugin support for a visual reading library
- Cover automation for book notes
- Daily note template for a simple journaling workflow
- Structured templates for different content types

## Getting Started

1. Open the vault in Obsidian.
2. Enable the recommended community plugins if needed.
3. Start by opening Bookshelf.md and adding your first book note.
4. Use Templates/ for new notes.

## Notes

This repository is intended as a personal knowledge-management template and may be customized further.
